DIARY PROJECT
프로젝트 설명
Java의 Calendar API를 이용하여 달력을 만들어 다이어리를 구성
다이어리에 일정을 등록, 수정, 삭제 할 수 있음
공지사항을 등록, 수정, 삭제 할 수 있음
개발환경
OS : Window11
Library : BootStrap5
Language : Java(JavaSE-17), HTML, CSS
사용 프로그램 : eclipse(2022-12 (4.26)), HeidiSQL(11.3.0)
Database : Mariadb(10.5.19)
server : tomcat10.1
홈으로 공지리스트 일정리스트
